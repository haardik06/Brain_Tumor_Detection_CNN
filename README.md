# Brain Tumor Detection using Convolutional Neural Network (CNN)

## Overview

This project detects brain tumors from MRI images using a Convolutional Neural Network (CNN). The model classifies MRI scans into one of four categories.

- Glioma
- Meningioma
- Pituitary Tumor
- No Tumor

---

## Features

- Brain MRI Classification
- Deep Learning using CNN
- TensorFlow/Keras
- Automatic Image Preprocessing
- Model Evaluation
- Confusion Matrix
- Classification Report
- Accuracy & Loss Graphs
- Single Image Prediction
